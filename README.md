# 1dv610-l3 Turn-based combat game

## Disclaimer
This App was created as part of a school assignment.

## Description
This is a simple web application where you can play a simple turn based game.

### Who is this for?
The app lets end users enjoy a simple game.
It shows module users how the module can be used.
The app shows clean code implementation, a test plan and a public deployment.

## How to run
1. Clone the repo
2. Move into the folder
3. Run npm install
4. Launch using live server extension
